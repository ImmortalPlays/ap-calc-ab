Cheat Sheet

The formulas you'll reach for most — limits, derivatives, integrals and the big theorems, all on one page.

Limits

\(\displaystyle\lim_{x\to 0}\frac{\sin x}{x} = 1\)
\(\displaystyle\lim_{x\to 0}\frac{1-\cos x}{x} = 0\)
Indeterminate forms to fix: \(\frac{0}{0},\ \frac{\infty}{\infty}\)
Continuous at \(a\) if \(\displaystyle\lim_{x\to a} f(x) = f(a)\)

Derivative rules

Power: \(\dfrac{d}{dx}\left[x^n\right] = n x^{n-1}\)
Product: \((fg)' = f'g + fg'\)
Quotient: \(\left(\dfrac{f}{g}\right)' = \dfrac{f'g - fg'}{g^2}\)
Chain: \(\dfrac{d}{dx} f(g(x)) = f'(g(x))\,g'(x)\)

Common derivatives

\(\dfrac{d}{dx}\sin x = \cos x,\quad \dfrac{d}{dx}\cos x = -\sin x\)
\(\dfrac{d}{dx}\tan x = \sec^2 x\)
\(\dfrac{d}{dx} e^x = e^x,\quad \dfrac{d}{dx}\ln x = \dfrac{1}{x}\)
\(\dfrac{d}{dx}\arctan x = \dfrac{1}{1+x^2}\)

Integral rules

\(\displaystyle\int x^n\,dx = \dfrac{x^{n+1}}{n+1} + C\quad(n\neq -1)\)
\(\displaystyle\int \dfrac{1}{x}\,dx = \ln|x| + C\)
\(\displaystyle\int e^x\,dx = e^x + C\)
\(\displaystyle\int \cos x\,dx = \sin x + C\)

The big theorems

FTC Part 1: \(\dfrac{d}{dx}\displaystyle\int_a^x f(t)\,dt = f(x)\)
FTC Part 2: \(\displaystyle\int_a^b f(x)\,dx = F(b) - F(a)\)
MVT: some \(c\) with \(f'(c) = \dfrac{f(b)-f(a)}{b-a}\)

Good to memorize

Avg rate of change \(= \dfrac{f(b)-f(a)}{b-a}\)
Avg value \(= \dfrac{1}{b-a}\displaystyle\int_a^b f(x)\,dx\)
Position → velocity → acceleration by differentiating
\(\displaystyle\int_a^b f(x)\,dx\) = signed area under the curve
