Unit 1 · Limits & Continuity

P1. Evaluate \(\displaystyle\lim_{x \to 3}\frac{x^2 - 9}{x^2 - 2x - 3}\).

▶Show solution
  1. Direct substitution gives \(\frac{0}{0}\) — factor both.
  2. \(\dfrac{(x-3)(x+3)}{(x-3)(x+1)}\); cancel \((x-3)\): \(\dfrac{x+3}{x+1}\).
  3. Substitute \(x = 3\): \(\dfrac{6}{4}\).
  4. Answer: \(\;\boxed{\tfrac{3}{2}}\)

P2. Find \(\displaystyle\lim_{x \to \infty}\frac{4x^2 - 3}{2x^2 + x}\) and identify the horizontal asymptote of \(f(x) = \dfrac{4x^2-3}{2x^2+x}\).

▶Show solution
  1. Same degree (both \(x^2\)) → compare leading coefficients.
  2. Divide every term by \(x^2\): \(\dfrac{4 - 3/x^2}{2 + 1/x}\).
  3. As \(x \to \infty\) the small terms vanish: \(\dfrac{4}{2}\).
  4. Answer: limit \(= \boxed{2}\); horizontal asymptote \(y = 2\) (the dashed line).

Unit 2 · Differentiation: Definition & Basic Rules

P1. For \(f(x) = x^3 - 4x^2 + 2x - 7\), find \(f'(x)\) and then \(f'(2)\).

▶Show solution
  1. Power rule term by term: \(f'(x) = 3x^2 - 8x + 2\).
  2. Evaluate at \(x = 2\): \(3(4) - 8(2) + 2 = 12 - 16 + 2\).
  3. Answer: \(f'(x) = 3x^2 - 8x + 2\) and \(\;\boxed{f'(2) = -2}\)

P2. Find the equation of the tangent line to \(f(x) = x^3\) at \(x = 1\).

▶Show solution
  1. \(f'(x) = 3x^2\), so the slope at \(x = 1\) is \(f'(1) = 3\).
  2. Point on the curve: \((1, f(1)) = (1, 1)\).
  3. Point-slope: \(y - 1 = 3(x - 1)\).
  4. Answer: \(\;\boxed{y = 3x - 2}\) (the red tangent line).

Unit 3 · Composite, Implicit & Inverse

P1. Differentiate \(y = e^{\sin x}\).

▶Show solution
  1. Chain rule: outside \(e^{(\cdot)}\), inside \(\sin x\).
  2. \(\dfrac{dy}{dx} = e^{\sin x}\cdot \dfrac{d}{dx}(\sin x)\).
  3. \(\dfrac{d}{dx}\sin x = \cos x\).
  4. Answer: \(\;\boxed{\dfrac{dy}{dx} = \cos x \cdot e^{\sin x}}\)

P2. Use implicit differentiation to find \(\dfrac{dy}{dx}\) for \(x^3 + y^3 = 6xy\).

▶Show solution
  1. Differentiate in \(x\); the right side needs the product rule: \(3x^2 + 3y^2\dfrac{dy}{dx} = 6y + 6x\dfrac{dy}{dx}\).
  2. Gather \(\dfrac{dy}{dx}\) terms: \(3y^2\dfrac{dy}{dx} - 6x\dfrac{dy}{dx} = 6y - 3x^2\).
  3. Factor: \(\dfrac{dy}{dx}(3y^2 - 6x) = 6y - 3x^2\).
  4. Answer: \(\;\boxed{\dfrac{dy}{dx} = \dfrac{2y - x^2}{y^2 - 2x}}\) (dividing through by 3).

Unit 4 · Contextual Applications

P1. A spherical snowball melts so its volume decreases at \(\dfrac{dV}{dt} = -2\ \text{cm}^3/\text{min}\). How fast is the radius shrinking when \(r = 3\) cm?

▶Show solution
  1. \(V = \tfrac{4}{3}\pi r^3 \Rightarrow \dfrac{dV}{dt} = 4\pi r^2 \dfrac{dr}{dt}\).
  2. Substitute \(\dfrac{dV}{dt} = -2\), \(r = 3\): \(-2 = 4\pi(9)\dfrac{dr}{dt}\).
  3. Solve: \(\dfrac{dr}{dt} = \dfrac{-2}{36\pi}\).
  4. Answer: \(\;\boxed{\dfrac{dr}{dt} = -\dfrac{1}{18\pi}\ \text{cm/min}}\) (negative = shrinking).

P2. Evaluate \(\displaystyle\lim_{x \to 0}\frac{e^x - 1}{x}\) using L'Hôpital's Rule.

▶Show solution
  1. Substitution gives \(\frac{0}{0}\) — L'Hôpital applies.
  2. Differentiate top and bottom: \(\dfrac{(e^x - 1)'}{(x)'} = \dfrac{e^x}{1}\).
  3. Substitute \(x = 0\): \(e^0 = 1\).
  4. Answer: \(\;\boxed{1}\)

Unit 5 · Analytical Applications

P1. Find and classify all local extrema of \(f(x) = x^4 - 8x^2\).

▶Show solution
  1. \(f'(x) = 4x^3 - 16x = 4x(x-2)(x+2)\); critical points \(x = -2, 0, 2\).
  2. \(f''(x) = 12x^2 - 16\).
  3. \(f''(0) = -16 < 0\) → local max at \((0, 0)\).
  4. \(f''(\pm 2) = 32 > 0\) → local min at \((\pm 2, -16)\).
  5. Answer: \(\;\boxed{\text{max } (0,0);\ \text{mins } (\pm 2, -16)}\) — the "W" shape.

P2. Find the absolute maximum and minimum of \(f(x) = x^3 - 12x\) on \([0, 3]\).

▶Show solution
  1. \(f'(x) = 3x^2 - 12 = 3(x-2)(x+2)\); critical points \(x = \pm 2\). Only \(x = 2\) is in \([0,3]\).
  2. Candidates test — evaluate at \(x = 0, 2, 3\):
  3. \(f(0) = 0\), \(f(2) = 8 - 24 = -16\), \(f(3) = 27 - 36 = -9\).
  4. Answer: \(\;\boxed{\text{abs max } 0 \text{ at } x=0,\ \text{abs min } -16 \text{ at } x=2}\)

Unit 6 · Integration & Accumulation

P1. Evaluate \(\displaystyle\int_1^3 (2x + 1)\,dx\).

▶Show solution
  1. Antiderivative: \(x^2 + x\).
  2. Evaluate top minus bottom: \((3^2 + 3) - (1^2 + 1)\).
  3. \(= 12 - 2\).
  4. Answer: \(\;\boxed{10}\)

P2. Evaluate \(\displaystyle\int x\sqrt{x^2 + 1}\,dx\).

▶Show solution
  1. Let \(u = x^2 + 1\), so \(du = 2x\,dx\), meaning \(x\,dx = \tfrac{1}{2}\,du\).
  2. Rewrite: \(\displaystyle\int \sqrt{u}\cdot\tfrac{1}{2}\,du = \tfrac{1}{2}\int u^{1/2}\,du\).
  3. Integrate: \(\tfrac{1}{2}\cdot\dfrac{u^{3/2}}{3/2} = \tfrac{1}{3}u^{3/2}\).
  4. Substitute back: \(\;\boxed{\tfrac{1}{3}(x^2 + 1)^{3/2} + C}\)

Unit 7 · Differential Equations

P1. Solve the initial value problem \(\dfrac{dy}{dx} = 3y,\ y(0) = 4\).

▶Show solution
  1. Recognize the growth model \(\dfrac{dy}{dx} = ky\) with \(k = 3\): \(y = y_0 e^{3x}\).
  2. Apply \(y(0) = 4\): \(4 = y_0 e^0 = y_0\).
  3. Answer: \(\;\boxed{y = 4e^{3x}}\)

P2. Solve \(\dfrac{dy}{dx} = \dfrac{x}{2y}\) with \(y(1) = 2\).

▶Show solution
  1. Separate: \(2y\,dy = x\,dx\).
  2. Integrate: \(y^2 = \dfrac{x^2}{2} + C\).
  3. Apply \(y(1) = 2\): \(4 = \dfrac{1}{2} + C \Rightarrow C = \dfrac{7}{2}\).
  4. Answer: \(\;\boxed{y = \sqrt{\dfrac{x^2}{2} + \dfrac{7}{2}}}\) (positive root, since \(y(1) = 2 > 0\)).

Unit 8 · Applications of Integration

P1. Find the area of the region bounded by \(y = 4 - x^2\) and the \(x\)-axis.

▶Show solution
  1. The curve meets the \(x\)-axis where \(4 - x^2 = 0\): \(x = -2\) and \(x = 2\).
  2. Area: \(\displaystyle\int_{-2}^{2} (4 - x^2)\,dx = \left[4x - \dfrac{x^3}{3}\right]_{-2}^{2}\).
  3. \(\left(8 - \dfrac{8}{3}\right) - \left(-8 + \dfrac{8}{3}\right) = \dfrac{16}{3} + \dfrac{16}{3}\).
  4. Answer: \(\;\boxed{\dfrac{32}{3}}\)

P2. The region under \(y = x^2\) from \(x = 0\) to \(x = 2\) is rotated about the \(x\)-axis. Find the volume.

▶Show solution
  1. Disks of radius \(R(x) = x^2\), so \([R(x)]^2 = x^4\).
  2. Set up: \(V = \pi\displaystyle\int_0^2 x^4\,dx\).
  3. Integrate: \(\pi\left[\dfrac{x^5}{5}\right]_0^2 = \pi\cdot\dfrac{32}{5}\).
  4. Answer: \(\;\boxed{\dfrac{32\pi}{5}}\)
