UNIT 515–18% of the exam

Analytical Applications of Differentiation

This is the biggest unit on the exam. Derivatives tell you the shape of a graph — where it rises, falls, curves, peaks, and bottoms out — which powers optimization and the major theorems.

1. Increasing, decreasing & critical points

Critical points are where \(f'(x) = 0\) or is undefined — the only places a graph can change direction. Between them, the sign of \(f'\) tells the direction:

  • \(f'(x) > 0\) → \(f\) is increasing
  • \(f'(x) < 0\) → \(f\) is decreasing

2. First derivative test

At a critical point, look at how \(f'\) changes sign:

  • \(+ \to -\): a local maximum.
  • \(- \to +\): a local minimum.
  • No sign change: neither (just a flat spot).

3. Concavity & the second derivative

  • \(f''(x) > 0\) → concave up (cup shape ∪)
  • \(f''(x) < 0\) → concave down (cap shape ∩)

Concavity is about whether the slope is itself increasing or decreasing.

4. Inflection points

An inflection point is where concavity changes sign — \(f''\) goes from \(+\) to \(-\) or vice versa. Candidates are where \(f''(x) = 0\) or is undefined, but you must confirm the sign actually flips.

5. Second derivative test

A shortcut for classifying a critical point \(c\) (where \(f'(c) = 0\)):

  • \(f''(c) > 0\) → concave up → local min.
  • \(f''(c) < 0\) → concave down → local max.
  • \(f''(c) = 0\) → inconclusive; fall back on the first derivative test.

6. Absolute extrema on a closed interval

To find the global max/min of a continuous \(f\) on \([a, b]\), use the Candidates Test: evaluate \(f\) at every critical point inside the interval and at both endpoints, then pick the largest and smallest \(y\)-values.

7. The Mean Value Theorem

If \(f\) is continuous on \([a, b]\) and differentiable on \((a, b)\), then there is some \(c\) where the instantaneous rate equals the average rate:

$$f'(c) = \frac{f(b) - f(a)}{b - a}$$

Geometrically: somewhere the tangent line is parallel to the secant through the endpoints. (Rolle's Theorem is the special case where \(f(a) = f(b)\), forcing \(f'(c) = 0\).)

8. Putting it together: curve sketching

A full analysis combines everything: use \(f'\) for increasing/decreasing and local extrema, and \(f''\) for concavity and inflection points. Together they pin down the exact shape of the graph.

9. Reading \(f\), \(f'\), and \(f''\) together

A huge share of exam points comes from translating between a function and its derivatives. Keep this dictionary straight:

  • \(f' > 0 \Rightarrow f\) increasing;   \(f' < 0 \Rightarrow f\) decreasing.
  • \(f'' > 0 \Rightarrow f\) concave up;   \(f'' < 0 \Rightarrow f\) concave down.
  • A local extremum of \(f\) sits where \(f'\) changes sign.
  • An inflection point of \(f\) sits where \(f''\) changes sign — i.e. at a local extremum of \(f'\).

So if you're handed the graph of \(f'\): count its sign changes to count \(f\)'s extrema, and its turning points to find \(f\)'s inflection points.

▶Example 1 — Local extrema of \(f(x) = x^3 - 3x\)
  1. \(f'(x) = 3x^2 - 3 = 3(x-1)(x+1)\).
  2. Critical points where \(f' = 0\): \(x = -1\) and \(x = 1\).
  3. Sign of \(f'\): positive for \(x < -1\), negative between, positive for \(x > 1\).
  4. So \(f'\) goes \(+ \to -\) at \(x = -1\) (local max) and \(- \to +\) at \(x = 1\) (local min).
  5. Answer: \(\;\boxed{\text{local max } (-1, 2),\ \text{local min } (1, -2)}\)

The crest on the left is the local max; the trough on the right is the local min.

▶Example 2 — Concavity & inflection of \(f(x) = x^3 - 3x\)
  1. \(f''(x) = 6x\).
  2. \(f'' < 0\) for \(x < 0\) (concave down); \(f'' > 0\) for \(x > 0\) (concave up).
  3. Concavity changes at \(x = 0\), and \(f(0) = 0\).
  4. Answer: \(\;\boxed{\text{inflection point at } (0, 0)}\)
▶Example 3 — Second derivative test (same \(f\))
  1. Critical points: \(x = -1\) and \(x = 1\); and \(f''(x) = 6x\).
  2. At \(x = -1\): \(f''(-1) = -6 < 0\) → concave down → local max.
  3. At \(x = 1\): \(f''(1) = 6 > 0\) → concave up → local min.
  4. Answer: \(\;\boxed{\text{max at } x=-1,\ \text{min at } x=1}\) — matches the first derivative test.
▶Example 4 — Absolute extrema of \(f(x) = x^3 - 3x\) on \([0, 3]\)
  1. Critical points in \([0,3]\): only \(x = 1\) (since \(x = -1\) is outside).
  2. Evaluate the candidates — critical point and endpoints:
  3. \(f(0) = 0\),   \(f(1) = -2\),   \(f(3) = 27 - 9 = 18\).
  4. Biggest is \(18\), smallest is \(-2\).
  5. Answer: \(\;\boxed{\text{abs max } 18 \text{ at } x=3,\ \text{abs min } -2 \text{ at } x=1}\)
▶Example 5 — Optimization: largest rectangle with perimeter 20
  1. Let width \(= x\). Perimeter \(20\) means the height is \(10 - x\).
  2. Area: \(A(x) = x(10 - x) = 10x - x^2\).
  3. Maximize: \(A'(x) = 10 - 2x = 0 \Rightarrow x = 5\).
  4. Then height \(= 10 - 5 = 5\), so the rectangle is a \(5 \times 5\) square.
  5. Answer: \(\;\boxed{\text{max area } = 25}\)
▶Example 6 — Optimization with a constraint: fencing against a wall

You have \(100\) ft of fence to make a rectangular pen against a long wall (the wall is the fourth side, so it needs no fence). Maximize the area.

  1. Let the two sides perpendicular to the wall be \(y\) and the side parallel be \(x\). Fence used: \(x + 2y = 100\).
  2. Solve the constraint: \(x = 100 - 2y\).
  3. Area: \(A(y) = xy = (100 - 2y)y = 100y - 2y^2\).
  4. Maximize: \(A'(y) = 100 - 4y = 0 \Rightarrow y = 25\), so \(x = 50\).
  5. Answer: \(\;\boxed{\text{max area } = 50 \times 25 = 1250\ \text{ft}^2}\)
▶Example 7 — Mean Value Theorem: \(f(x) = x^2\) on \([0, 2]\)
  1. \(f\) is continuous and differentiable, so MVT applies.
  2. Average rate (secant slope): \(\dfrac{f(2) - f(0)}{2 - 0} = \dfrac{4 - 0}{2} = 2\).
  3. Set the instantaneous rate equal: \(f'(c) = 2c = 2\).
  4. Solve: \(c = 1\) (and it's inside \((0, 2)\), as required).
  5. Answer: \(\;\boxed{c = 1}\) — the tangent there is parallel to the secant.

The dashed secant through \((0,0)\) and \((2,4)\) has slope 2; the tangent at \((1,1)\) has the same slope.

▶Example 8 — Reading a graph of \(f'\): suppose \(f'(x) = x^2 - 4\)

The graph below is \(f'\) (the derivative), not \(f\). Describe where \(f\) increases, decreases, and has local extrema.

  1. \(f' = 0\) where \(x^2 - 4 = 0\): at \(x = -2\) and \(x = 2\) (where the curve crosses the axis).
  2. \(f' > 0\) for \(x < -2\) and \(x > 2\) (curve above axis) → \(f\) is increasing there.
  3. \(f' < 0\) for \(-2 < x < 2\) (curve below axis) → \(f\) is decreasing there.
  4. \(f'\) changes \(+\to-\) at \(x = -2\) → local max; \(-\to+\) at \(x = 2\) → local min.
  5. Answer: \(\;\boxed{\text{local max at } x=-2,\ \text{local min at } x=2}\)

Remember: the zeros of this curve are \(f\)'s critical points, and its own minimum (at \(x = 0\)) is an inflection point of \(f\).
