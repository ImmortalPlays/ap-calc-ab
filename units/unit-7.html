Differential Equations

A differential equation describes how something changes; solving it recovers the function itself. On the AB exam this means slope fields, separable equations, and the exponential growth/decay model.

1. What is a differential equation?

A differential equation involves a derivative, like \(\dfrac{dy}{dx} = xy\). A solution is a function \(y\) that makes it true. You can verify a proposed solution by differentiating it and checking that both sides match.

2. Slope fields

A slope field draws a tiny line segment at each point with slope \(\dfrac{dy}{dx}\). It lets you see the family of solution curves without solving anything: a particular solution flows through the field following the segments, like a stream following arrows.

3. Separable equations

If you can get all the \(y\)'s on one side and all the \(x\)'s on the other, integrate both sides:

$$\frac{dy}{dx} = g(x)h(y) \;\Rightarrow\; \int \frac{1}{h(y)}\,dy = \int g(x)\,dx$$

Don't forget a single \(+C\) — and add it before solving for \(y\).

4. Initial value problems

An initial condition like \(y(0) = 100\) pins down the constant \(C\), turning the whole family of solutions into one particular solution. Solve generally first, then use the condition to find \(C\).

5. Exponential growth & decay

The classic model: a quantity whose rate of change is proportional to itself.

$$\frac{dy}{dt} = ky \;\Rightarrow\; y = y_0\,e^{kt}$$

\(k > 0\) is growth; \(k < 0\) is decay; \(y_0\) is the starting amount. Recognize this form instantly — you can write the answer without re-deriving it.

6. Half-life & doubling

For decay, the half-life is the time to drop to half. Since each half-life multiplies by \(\tfrac12\), the amount after \(n\) half-lives is \(y_0\left(\tfrac12\right)^n\). The same idea (with \(\times 2\)) gives doubling time for growth.

▶Example 1 — Verify a solution: does \(y = e^{2x}\) solve \(y' = 2y\)?
  1. Differentiate the candidate: \(y' = \dfrac{d}{dx}e^{2x} = 2e^{2x}\).
  2. Compute the right side: \(2y = 2e^{2x}\).
  3. Both sides equal \(2e^{2x}\).
  4. Answer: \(\;\boxed{\text{yes, } y = e^{2x} \text{ is a solution}}\).
▶Example 2 — Separable equation: \(\dfrac{dy}{dx} = xy\)
  1. Separate variables: \(\dfrac{1}{y}\,dy = x\,dx\).
  2. Integrate both sides: \(\int \dfrac{1}{y}\,dy = \int x\,dx\).
  3. This gives \(\ln|y| = \dfrac{x^2}{2} + C\).
  4. Exponentiate: \(|y| = e^{x^2/2 + C} = e^C e^{x^2/2}\).
  5. Answer: \(\;\boxed{y = A\,e^{x^2/2}}\) where \(A = \pm e^C\) is a constant.
▶Example 3 — Initial value problem: \(\dfrac{dy}{dx} = \dfrac{x}{y},\ y(0) = 3\)
  1. Separate: \(y\,dy = x\,dx\).
  2. Integrate: \(\dfrac{y^2}{2} = \dfrac{x^2}{2} + C\), so \(y^2 = x^2 + C_1\).
  3. Apply \(y(0) = 3\): \(3^2 = 0 + C_1 \Rightarrow C_1 = 9\).
  4. So \(y^2 = x^2 + 9\); take the positive root (since \(y(0) = 3 > 0\)).
  5. Answer: \(\;\boxed{y = \sqrt{x^2 + 9}}\)
▶Example 4 — Exponential growth: \(\dfrac{dy}{dt} = 0.5y,\ y(0) = 100\)
  1. This is the growth model with \(k = 0.5\), so \(y = y_0 e^{0.5t}\).
  2. Use the initial condition \(y(0) = 100\): \(100 = y_0 e^0 = y_0\).
  3. So \(y_0 = 100\).
  4. Answer: \(\;\boxed{y = 100\,e^{0.5t}}\)

The solution curve starts at \(100\) and grows faster and faster — the signature of exponential growth.

▶Example 5 — Exponential decay: \(\dfrac{dy}{dt} = -0.5y,\ y(0) = 100\)
  1. Same model, now with \(k = -0.5\): \(y = y_0 e^{-0.5t}\).
  2. Initial condition \(y(0) = 100\) gives \(y_0 = 100\).
  3. Answer: \(\;\boxed{y = 100\,e^{-0.5t}}\)
  4. The quantity decays toward \(0\) but never quite reaches it.

Mirror image of growth: it starts at \(100\) and eases down toward the \(t\)-axis.

▶Example 6 — Half-life: a sample starts at 100 mg with a 10-year half-life
  1. Each half-life multiplies the amount by \(\tfrac12\): \(y = 100\left(\tfrac12\right)^{t/10}\).
  2. After \(10\) years (one half-life): \(100 \cdot \tfrac12 = 50\) mg.
  3. After \(20\) years (two half-lives): \(50 \cdot \tfrac12 = 25\) mg.
  4. Answer: \(\;\boxed{25 \text{ mg remain after 20 years}}\).
