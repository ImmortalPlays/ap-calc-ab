1. What is a differential equation?

A differential equation is an equation involving a derivative, like \(\dfrac{dy}{dx} = xy\). A solution is a function \(y\) that makes it true. A slope field sketches the slope \(\dfrac{dy}{dx}\) at many points so you can see the shape of solutions.

2. Separable equations

If you can get all the \(y\)’s on one side and all the \(x\)’s on the other, integrate both sides:

$$\frac{dy}{dx} = g(x)h(y) \;\Rightarrow\; \int \frac{1}{h(y)}\,dy = \int g(x)\,dx$$

3. Exponential growth & decay

The classic model: a quantity whose rate of change is proportional to itself.

$$\frac{dy}{dt} = ky \;\Rightarrow\; y = y_0\,e^{kt}$$

\(k > 0\) is growth; \(k < 0\) is decay; \(y_0\) is the starting amount.

Step-by-step solutions

Click any example to expand the full solution.

Example 1 — Separable equation: \(\dfrac{dy}{dx} = xy\)
  1. Separate variables: \(\dfrac{1}{y}\,dy = x\,dx\).
  2. Integrate both sides: \(\int \dfrac{1}{y}\,dy = \int x\,dx\).
  3. This gives \(\ln|y| = \dfrac{x^2}{2} + C\).
  4. Exponentiate: \(|y| = e^{x^2/2 + C} = e^C e^{x^2/2}\).
  5. Answer: \(\;\boxed{y = A\,e^{x^2/2}}\) where \(A = \pm e^C\) is a constant.
Example 2 — Exponential growth: \(\dfrac{dy}{dt} = 0.5y,\ y(0) = 100\)
  1. This is the growth model with \(k = 0.5\), so \(y = y_0 e^{0.5t}\).
  2. Use the initial condition \(y(0) = 100\): \(100 = y_0 e^0 = y_0\).
  3. So \(y_0 = 100\).
  4. Answer: \(\;\boxed{y = 100\,e^{0.5t}}\)

The solution curve starts at \(100\) and grows faster and faster — the signature of exponential growth.
