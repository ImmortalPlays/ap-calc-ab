Applications of Integration

Integrals add up area, volume, and total change. The hardest part isn't the integration — it's setting up the right integral. Draw the picture every time.

1. Area under a curve

The area between \(y = f(x)\) and the \(x\)-axis from \(a\) to \(b\) is \(\displaystyle\int_a^b f(x)\,dx\) — when \(f\ge 0\). If the curve dips below the axis and you want geometric area, integrate \(|f(x)|\) (split at the zeros).

2. Area between two curves (in \(x\))

The area between a top curve and a bottom curve from \(a\) to \(b\):

$$\int_a^b \big[\,\text{top}(x) - \text{bottom}(x)\,\big]\,dx$$

If the curves cross, find the intersection points first — those become your limits (and you may need to split the region).

3. Area integrating in \(y\)

Sometimes it's cleaner to slice horizontally. Then integrate right curve minus left curve with respect to \(y\):

$$\int_c^d \big[\,\text{right}(y) - \text{left}(y)\,\big]\,dy$$

4. Volume by disks

Rotate a region around an axis and slice into thin disks of radius \(R\):

$$V = \pi\int_a^b \big[R(x)\big]^2\,dx$$

5. Volume by washers

If there's a gap between the region and the axis, each slice is a washer (a disk with a hole) — outer radius \(R\), inner radius \(r\):

$$V = \pi\int_a^b \Big(\big[R(x)\big]^2 - \big[r(x)\big]^2\Big)\,dx$$

6. Volume by known cross-sections

No rotation needed: if each slice perpendicular to the axis is a known shape (square, semicircle, triangle), integrate the slice's area. For squares with side \(s(x)\): \(V = \displaystyle\int_a^b \big[s(x)\big]^2\,dx\).

7. Average value

The average height of \(f\) over \([a,b]\): \(f_{\text{avg}} = \dfrac{1}{b-a}\displaystyle\int_a^b f(x)\,dx\).

8. Motion: displacement vs. total distance

From a velocity function \(v(t)\):

  • Displacement (net change in position): \(\displaystyle\int_a^b v(t)\,dt\)
  • Total distance traveled: \(\displaystyle\int_a^b |v(t)|\,dt\)

This is the Net Change Theorem: integrating a rate gives the total accumulated change.

Step-by-step solutions

▶Example 1 — Area between \(y = x\) and \(y = x^2\) from 0 to 1
  1. On \([0, 1]\), the line \(y = x\) sits above the parabola \(y = x^2\), so top \(-\) bottom \(= x - x^2\).
  2. Set up: \(\displaystyle\int_0^1 (x - x^2)\,dx\).
  3. Integrate: \(\left[\dfrac{x^2}{2} - \dfrac{x^3}{3}\right]_0^1\).
  4. Evaluate: \(\dfrac{1}{2} - \dfrac{1}{3} = \dfrac{3 - 2}{6}\).
  5. Answer: \(\;\boxed{\dfrac{1}{6}}\)

The shaded sliver between the line and the parabola from \(x = 0\) to \(x = 1\) has area \(\tfrac{1}{6}\).

▶Example 2 — Area needing intersection points: \(y = 2x\) and \(y = x^2\)
  1. Find where they meet: \(2x = x^2 \Rightarrow x^2 - 2x = 0 \Rightarrow x(x-2) = 0\), so \(x = 0\) and \(x = 2\).
  2. Between them the line \(2x\) is on top: top \(-\) bottom \(= 2x - x^2\).
  3. Set up: \(\displaystyle\int_0^2 (2x - x^2)\,dx = \left[x^2 - \dfrac{x^3}{3}\right]_0^2\).
  4. Evaluate: \(4 - \dfrac{8}{3} = \dfrac{12 - 8}{3}\).
  5. Answer: \(\;\boxed{\dfrac{4}{3}}\)
▶Example 3 — Average value of \(f(x) = x^2\) on \([0, 3]\)
  1. Use \(f_{\text{avg}} = \dfrac{1}{b-a}\displaystyle\int_a^b f\,dx\) with \(a = 0,\ b = 3\).
  2. \(\displaystyle\int_0^3 x^2\,dx = \left[\dfrac{x^3}{3}\right]_0^3 = 9\).
  3. Divide by \(3\): \(\dfrac{1}{3}(9)\).
  4. Answer: \(\;\boxed{3}\)
▶Example 4 — Volume by disks: \(y = \sqrt{x}\) about the \(x\)-axis, 0 to 4
  1. Radius of each disk is \(R(x) = \sqrt{x}\), so \([R(x)]^2 = x\).
  2. Set up: \(V = \pi\displaystyle\int_0^4 x\,dx\).
  3. Integrate: \(\pi\left[\dfrac{x^2}{2}\right]_0^4 = \pi \cdot \dfrac{16}{2}\).
  4. Answer: \(\;\boxed{8\pi}\)
▶Example 5 — Volume by washers: region between \(y = \sqrt{x}\) and \(y = x\), about the \(x\)-axis
  1. The curves meet at \(x = 0\) and \(x = 1\); on \((0,1)\), \(\sqrt{x}\) is the outer radius and \(x\) is the inner radius.
  2. Washer area: \(\pi\big[(\sqrt{x})^2 - (x)^2\big] = \pi\big[x - x^2\big]\).
  3. Set up: \(V = \pi\displaystyle\int_0^1 (x - x^2)\,dx = \pi\left[\dfrac{x^2}{2} - \dfrac{x^3}{3}\right]_0^1\).
  4. Evaluate the bracket: \(\dfrac{1}{2} - \dfrac{1}{3} = \dfrac{1}{6}\).
  5. Answer: \(\;\boxed{\dfrac{\pi}{6}}\)

The solid is what you get spinning the sliver between these two curves around the \(x\)-axis — each slice is a washer.

▶Example 6 — Known cross-sections: squares on a base under \(y = \sqrt{x}\)

The base is the region under \(y = \sqrt{x}\) from \(0\) to \(4\). Each slice perpendicular to the \(x\)-axis is a square. Find the volume.

  1. The side of each square equals the height of the region: \(s(x) = \sqrt{x}\).
  2. Square's area: \([s(x)]^2 = (\sqrt{x})^2 = x\).
  3. Set up: \(V = \displaystyle\int_0^4 x\,dx = \left[\dfrac{x^2}{2}\right]_0^4 = \dfrac{16}{2}\).
  4. Answer: \(\;\boxed{8}\) (note: no \(\pi\) — there's no rotation here).
▶Example 7 — Displacement vs. total distance: \(v(t) = t^2 - 4\) on \([0, 3]\)
  1. Displacement \(= \displaystyle\int_0^3 (t^2 - 4)\,dt = \left[\dfrac{t^3}{3} - 4t\right]_0^3 = 9 - 12 = -3\).
  2. For distance, find where \(v = 0\): \(t^2 = 4 \Rightarrow t = 2\). \(v < 0\) on \([0,2]\), \(v > 0\) on \([2,3]\).
  3. \(\displaystyle\int_0^2 (4 - t^2)\,dt = \left[4t - \dfrac{t^3}{3}\right]_0^2 = 8 - \dfrac{8}{3} = \dfrac{16}{3}\).
  4. \(\displaystyle\int_2^3 (t^2 - 4)\,dt = \left[\dfrac{t^3}{3} - 4t\right]_2^3 = (-3) - \left(-\dfrac{16}{3}\right) = \dfrac{7}{3}\).
  5. Answer: \(\;\boxed{\text{displacement } -3,\ \text{total distance } \dfrac{16}{3} + \dfrac{7}{3} = \dfrac{23}{3}}\)
