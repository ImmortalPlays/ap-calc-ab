UNIT 810–15% of the exam

Applications of Integration

1. Area between two curves

The area between a top curve and a bottom curve from \(a\) to \(b\):

$$\int_a^b \big[\,\text{top}(x) - \text{bottom}(x)\,\big]\,dx$$

2. Average value of a function

The average height of \(f\) over \([a, b]\):

$$f_{\text{avg}} = \frac{1}{b - a}\int_a^b f(x)\,dx$$

3. Volumes of revolution (disks)

Rotate a region around the \(x\)-axis and you sweep out a solid made of thin disks of radius \(R(x)\):

$$V = \pi\int_a^b \big[R(x)\big]^2\,dx$$

4. Motion: displacement vs. distance

  • Displacement: \(\displaystyle\int_a^b v(t)\,dt\) (net change in position)
  • Total distance: \(\displaystyle\int_a^b |v(t)|\,dt\)

▶Example 1 — Area between \(y = x\) and \(y = x^2\) from 0 to 1
  1. On \([0, 1]\), the line \(y = x\) sits above the parabola \(y = x^2\), so top \(-\) bottom \(= x - x^2\).
  2. Set up: \(\displaystyle\int_0^1 (x - x^2)\,dx\).
  3. Integrate: \(\left[\dfrac{x^2}{2} - \dfrac{x^3}{3}\right]_0^1\).
  4. Evaluate: \(\dfrac{1}{2} - \dfrac{1}{3} = \dfrac{3 - 2}{6}\).
  5. Answer: \(\;\boxed{\dfrac{1}{6}}\)

The shaded sliver between the line and the parabola from \(x = 0\) to \(x = 1\) has area \(\tfrac{1}{6}\).

▶Example 2 — Average value of \(f(x) = x^2\) on \([0, 3]\)
  1. Use \(f_{\text{avg}} = \dfrac{1}{b-a}\displaystyle\int_a^b f(x)\,dx\) with \(a = 0,\ b = 3\).
  2. \(\displaystyle\int_0^3 x^2\,dx = \left[\dfrac{x^3}{3}\right]_0^3 = \dfrac{27}{3} = 9\).
  3. Divide by \(b - a = 3\): \(\dfrac{1}{3}(9)\).
  4. Answer: \(\;\boxed{3}\)
▶Example 3 — Volume by disks: \(y = \sqrt{x}\) rotated about the \(x\)-axis, 0 to 4
  1. Radius of each disk is \(R(x) = \sqrt{x}\), so \([R(x)]^2 = x\).
  2. Set up: \(V = \pi\displaystyle\int_0^4 x\,dx\).
  3. Integrate: \(\pi\left[\dfrac{x^2}{2}\right]_0^4 = \pi \cdot \dfrac{16}{2}\).
  4. Answer: \(\;\boxed{8\pi}\)
