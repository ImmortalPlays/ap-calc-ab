Unit 1: Limits & Continuity

← All units

1. What is a limit?

A limit describes the value a function approaches as the input gets close to some number. We write:

$$\lim_{x \to a} f(x) = L$$

This reads: “as \(x\) gets close to \(a\), \(f(x)\) gets close to \(L\).” Notice we care about what happens near \(a\), not necessarily at \(a\).

2. One-sided limits

You can approach \(a\) from the left or the right:

  • Left-hand limit: \(\lim_{x \to a^-} f(x)\)
  • Right-hand limit: \(\lim_{x \to a^+} f(x)\)

The two-sided limit exists only if both sides agree:

$$\lim_{x \to a^-} f(x) = \lim_{x \to a^+} f(x) = L$$

3. Evaluating limits

Start by plugging the number in (direct substitution). If you get a real number, that’s your answer. If you get the indeterminate form \(\frac{0}{0}\), you must rewrite the function first — usually by factoring or rationalizing.

4. Continuity

A function is continuous at \(x = a\) if all three of these are true:

  1. \(f(a)\) is defined
  2. \(\lim_{x \to a} f(x)\) exists
  3. \(\lim_{x \to a} f(x) = f(a)\)

If any one fails, there is a discontinuity. The three common types are removable (a hole), jump, and infinite (a vertical asymptote).

Step-by-step solutions

Example 1 — Direct substitution: \(\lim_{x \to 3}\,(2x^2 - 4x + 1)\)
  1. This is a polynomial, which is continuous everywhere, so we can substitute directly.
  2. Plug in \(x = 3\): \(\;2(3)^2 - 4(3) + 1\)
  3. Simplify: \(\;18 - 12 + 1\)
  4. Answer: \(\;\boxed{7}\)
Example 2 — Factoring a \(\tfrac{0}{0}\) form: \(\lim_{x \to 2}\dfrac{x^2 - 4}{x - 2}\)
  1. Try direct substitution: \(\dfrac{2^2 - 4}{2 - 2} = \dfrac{0}{0}\) — indeterminate, so we rewrite.
  2. Factor the numerator: \(x^2 - 4 = (x - 2)(x + 2)\).
  3. Cancel the common \((x - 2)\): \(\dfrac{(x-2)(x+2)}{x-2} = x + 2\).
  4. Now substitute \(x = 2\): \(\;2 + 2\).
  5. Answer: \(\;\boxed{4}\)
Example 3 — Rationalizing: \(\lim_{x \to 0}\dfrac{\sqrt{x+4} - 2}{x}\)
  1. Direct substitution gives \(\dfrac{\sqrt{4} - 2}{0} = \dfrac{0}{0}\) — indeterminate.
  2. Multiply top and bottom by the conjugate \(\sqrt{x+4} + 2\):
  3. $$\dfrac{\sqrt{x+4}-2}{x}\cdot\dfrac{\sqrt{x+4}+2}{\sqrt{x+4}+2} = \dfrac{(x+4) - 4}{x\left(\sqrt{x+4}+2\right)}$$
  4. Simplify the numerator to \(x\), then cancel: \(\dfrac{x}{x(\sqrt{x+4}+2)} = \dfrac{1}{\sqrt{x+4}+2}\).
  5. Substitute \(x = 0\): \(\dfrac{1}{\sqrt{4}+2} = \dfrac{1}{4}\).
  6. Answer: \(\;\boxed{\tfrac{1}{4}}\)
Example 4 — Checking continuity at a point

Let \(f(x) = \dfrac{x^2 - 1}{x - 1}\) for \(x \neq 1\), and \(f(1) = 3\). Is \(f\) continuous at \(x = 1\)?

  1. Is \(f(1)\) defined? Yes, \(f(1) = 3\).
  2. Does the limit exist? Factor: \(\dfrac{(x-1)(x+1)}{x-1} = x + 1\), so \(\lim_{x \to 1} f(x) = 2\).
  3. Do they match? The limit is \(2\) but \(f(1) = 3\), so \(2 \neq 3\).
  4. Conclusion: \(f\) is not continuous at \(x = 1\). It is a removable discontinuity (a hole).
