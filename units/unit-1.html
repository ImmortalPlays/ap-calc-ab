Unit 1: Limits & Continuity

← All units

1. What is a limit?

A limit describes the value a function approaches as the input gets close to some number. We write:

$$\lim_{x \to a} f(x) = L$$

This reads: “as \(x\) gets close to \(a\), \(f(x)\) gets close to \(L\).” Notice we care about what happens near \(a\), not necessarily at \(a\).

2. One-sided limits

You can approach \(a\) from the left or the right:

  • Left-hand limit (coming from values below \(a\)): \(\lim_{x \to a^-} f(x)\)
  • Right-hand limit (coming from values above \(a\)): \(\lim_{x \to a^+} f(x)\)

The two-sided limit exists only if both sides agree:

$$\lim_{x \to a^-} f(x) = \lim_{x \to a^+} f(x) = L$$

If the two sides disagree, the limit does not exist (DNE).

3. Evaluating limits

Start by plugging the number in (direct substitution). If you get a real number, that’s your answer. If you get the indeterminate form \(\frac{0}{0}\), you must rewrite the function first — usually by factoring or rationalizing.

4. Infinite limits & limits at infinity

Two different “infinity” situations show up:

  • Infinite limit: the function blows up near a point (like \(\frac{1}{x-2}\) at \(x = 2\)). The limit is \(\pm\infty\), the two-sided limit DNE, and you get a vertical asymptote.
  • Limit at infinity: \(x\) itself grows without bound. \(\lim_{x \to \pm\infty} f(x)\) describes the end behavior, which often levels off at a horizontal asymptote.

5. Continuity

A function is continuous at \(x = a\) if all three of these are true:

  1. \(f(a)\) is defined
  2. \(\lim_{x \to a} f(x)\) exists
  3. \(\lim_{x \to a} f(x) = f(a)\)

If any one fails, there is a discontinuity. The three common types are removable (a hole), jump, and infinite (a vertical asymptote).

Example 1 — Direct substitution: \(\lim_{x \to 3}\,(2x^2 - 4x + 1)\)
  1. This is a polynomial, which is continuous everywhere, so we can substitute directly.
  2. Plug in \(x = 3\): \(\;2(3)^2 - 4(3) + 1\)
  3. Simplify: \(\;18 - 12 + 1\)
  4. Answer: \(\;\boxed{7}\)
Example 2 — Factoring a \(\tfrac{0}{0}\) form: \(\lim_{x \to 2}\dfrac{x^2 - 4}{x - 2}\)
  1. Try direct substitution: \(\dfrac{2^2 - 4}{2 - 2} = \dfrac{0}{0}\) — indeterminate, so we rewrite.
  2. Factor the numerator: \(x^2 - 4 = (x - 2)(x + 2)\).
  3. Cancel the common \((x - 2)\): \(\dfrac{(x-2)(x+2)}{x-2} = x + 2\).
  4. Now substitute \(x = 2\): \(\;2 + 2\).
  5. Answer: \(\;\boxed{4}\)

The graph is the line \(y = x + 2\) with an open hole at \((2, 4)\) — the limit is \(4\) even though the function isn’t defined there.

Example 3 — Rationalizing: \(\lim_{x \to 0}\dfrac{\sqrt{x+4} - 2}{x}\)
  1. Direct substitution gives \(\dfrac{\sqrt{4} - 2}{0} = \dfrac{0}{0}\) — indeterminate.
  2. Multiply top and bottom by the conjugate \(\sqrt{x+4} + 2\):
  3. $$\dfrac{\sqrt{x+4}-2}{x}\cdot\dfrac{\sqrt{x+4}+2}{\sqrt{x+4}+2} = \dfrac{(x+4) - 4}{x\left(\sqrt{x+4}+2\right)}$$
  4. Simplify the numerator to \(x\), then cancel: \(\dfrac{x}{x(\sqrt{x+4}+2)} = \dfrac{1}{\sqrt{x+4}+2}\).
  5. Substitute \(x = 0\): \(\dfrac{1}{\sqrt{4}+2} = \dfrac{1}{4}\).
  6. Answer: \(\;\boxed{\tfrac{1}{4}}\)
Example 4 — One-sided limits & a jump: \(f(x)=\begin{cases}x^2 & x<1\\ x+1 & x\ge 1\end{cases}\)
  1. Approach from the left (use the \(x^2\) piece): \(\lim_{x \to 1^-} f(x) = 1^2 = 1\).
  2. Approach from the right (use the \(x+1\) piece): \(\lim_{x \to 1^+} f(x) = 1 + 1 = 2\).
  3. The two sides disagree: \(1 \neq 2\).
  4. Conclusion: \(\lim_{x \to 1} f(x)\) does not exist — a jump discontinuity.

Notice the open circle at \((1, 1)\) where the left piece stops, and the filled dot at \((1, 2)\) where the right piece starts. The graph literally jumps.

Example 5 — Infinite limit (approaching from both sides): \(\lim_{x \to 2}\dfrac{1}{x-2}\)
  1. Substituting gives \(\dfrac{1}{0}\) — undefined, so check each side separately.
  2. From the left (\(x\) slightly less than 2): the bottom is a tiny negative number, so \(\lim_{x \to 2^-}\dfrac{1}{x-2} = -\infty\).
  3. From the right (\(x\) slightly more than 2): the bottom is a tiny positive number, so \(\lim_{x \to 2^+}\dfrac{1}{x-2} = +\infty\).
  4. The sides head to opposite infinities, so the two-sided limit does not exist. There is a vertical asymptote at \(x = 2\).

Watch the curve dive to \(-\infty\) just left of the dashed line and rocket to \(+\infty\) just right of it.

Example 6 — Limit at infinity: \(\lim_{x \to \infty}\dfrac{3x^2 + 2}{x^2 + 1}\)
  1. For a rational function, compare the highest powers — here both are \(x^2\).
  2. Divide every term by \(x^2\): \(\dfrac{3 + 2/x^2}{1 + 1/x^2}\).
  3. As \(x \to \infty\), the \(2/x^2\) and \(1/x^2\) terms shrink to \(0\): \(\dfrac{3 + 0}{1 + 0}\).
  4. Answer: \(\;\boxed{3}\). The line \(y = 3\) is a horizontal asymptote.

Zoom out: the curve flattens toward the dashed line \(y = 3\) in both directions.

Example 7 — Checking continuity at a point

Let \(f(x) = \dfrac{x^2 - 1}{x - 1}\) for \(x \neq 1\), and \(f(1) = 3\). Is \(f\) continuous at \(x = 1\)?

  1. Is \(f(1)\) defined? Yes, \(f(1) = 3\).
  2. Does the limit exist? Factor: \(\dfrac{(x-1)(x+1)}{x-1} = x + 1\), so \(\lim_{x \to 1} f(x) = 2\).
  3. Do they match? The limit is \(2\) but \(f(1) = 3\), so \(2 \neq 3\).
  4. Conclusion: \(f\) is not continuous at \(x = 1\). It is a removable discontinuity (a hole).

The open hole sits at \((1, 2)\) where the curve naturally goes; the red dot is the separately-defined \(f(1) = 3\) floating above it.
