UNIT 617–20% of the exam

Integration & Accumulation of Change

Integration is differentiation run backwards — and also the way we add up infinitely many tiny pieces to get area and total change. This is the largest unit on the exam, anchored by the Fundamental Theorem of Calculus.

1. Antiderivatives & the indefinite integral

Integration undoes differentiation. The power rule for integrals:

$$\int x^n\,dx = \frac{x^{n+1}}{n+1} + C \quad (n \neq -1)$$

The \(+C\) matters — any constant disappears when you differentiate, so we add it back to capture all antiderivatives.

2. Basic antiderivative rules

  • \(\displaystyle\int e^x\,dx = e^x + C\)
  • \(\displaystyle\int \frac{1}{x}\,dx = \ln|x| + C\)
  • \(\displaystyle\int \cos x\,dx = \sin x + C\),   \(\displaystyle\int \sin x\,dx = -\cos x + C\)
  • \(\displaystyle\int \sec^2 x\,dx = \tan x + C\)

3. Riemann sums

Before the shortcut, the definite integral was defined by adding up rectangles. With \(n\) strips of width \(\Delta x\):

  • Left sum: heights from the left edge of each strip.
  • Right sum: heights from the right edge.
  • Midpoint and trapezoid rules: usually more accurate.

As \(n \to \infty\) (\(\Delta x \to 0\)), the sum becomes the exact integral.

4. The definite integral as signed area

\(\displaystyle\int_a^b f(x)\,dx\) is the signed area between the curve and the \(x\)-axis from \(a\) to \(b\) — area above the axis counts positive, below counts negative. Useful properties:

  • \(\displaystyle\int_a^a f = 0\)   and   \(\displaystyle\int_b^a f = -\int_a^b f\)
  • \(\displaystyle\int_a^b f + \int_b^c f = \int_a^c f\)

5. Fundamental Theorem, Part 1 (accumulation)

An accumulation function \(g(x) = \displaystyle\int_a^x f(t)\,dt\) has derivative equal to the integrand:

$$\frac{d}{dx}\int_a^x f(t)\,dt = f(x)$$

If the upper limit is itself a function, tack on the chain rule: \(\dfrac{d}{dx}\displaystyle\int_a^{u(x)} f(t)\,dt = f(u(x))\cdot u'(x)\).

6. Fundamental Theorem, Part 2 (evaluation)

This is how you actually compute a definite integral:

$$\int_a^b f(x)\,dx = F(b) - F(a)$$

where \(F\) is any antiderivative of \(f\). Find an antiderivative, plug in the top and bottom, subtract.

7. \(u\)-substitution

The reverse of the chain rule. Pick an inside function \(u\), compute \(du\), and rewrite the whole integral in terms of \(u\) before integrating. For definite integrals, either convert the limits to \(u\)-values or switch back to \(x\) before plugging in.

8. Average value of a function

The average height of \(f\) over \([a, b]\):

$$f_{\text{avg}} = \frac{1}{b - a}\int_a^b f(x)\,dx$$

▶Example 1 — Indefinite integral: \(\displaystyle\int (3x^2 + 2x)\,dx\)
  1. Integrate term by term using the power rule.
  2. \(\int 3x^2\,dx = 3 \cdot \dfrac{x^3}{3} = x^3\).
  3. \(\int 2x\,dx = 2 \cdot \dfrac{x^2}{2} = x^2\).
  4. Add the constant of integration.
  5. Answer: \(\;\boxed{x^3 + x^2 + C}\)
▶Example 2 — Antiderivatives with trig & \(e^x\): \(\displaystyle\int (\cos x + e^x)\,dx\)
  1. Integrate each piece using the known rules.
  2. \(\int \cos x\,dx = \sin x\).
  3. \(\int e^x\,dx = e^x\).
  4. Answer: \(\;\boxed{\sin x + e^x + C}\)
▶Example 3 — Left Riemann sum from a table

Water flows into a tank at rate \(r(t)\) gal/min. Estimate the gallons added over \([0, 4]\) with a left sum, \(\Delta t = 1\):

  1. Table: \(r(0)=2,\ r(1)=5,\ r(2)=7,\ r(3)=8,\ r(4)=10\).
  2. A left sum uses the value at the left end of each 1-minute strip: \(r(0), r(1), r(2), r(3)\).
  3. Sum \(\times \Delta t\): \((2 + 5 + 7 + 8)(1)\).
  4. Answer: \(\;\boxed{22 \text{ gallons (estimate)}}\)
▶Example 4 — Definite integral via FTC: \(\displaystyle\int_0^2 3x^2\,dx\)
  1. Find an antiderivative: \(F(x) = x^3\).
  2. Apply Part 2: \(F(2) - F(0)\).
  3. \(= 2^3 - 0^3 = 8 - 0\).
  4. Answer: \(\;\boxed{8}\) — the area under \(y = 3x^2\) from 0 to 2.

The region under \(y = 3x^2\) between the dashed lines \(x = 0\) and \(x = 2\) has area 8.

▶Example 5 — Signed area: \(\displaystyle\int_{-1}^{1} x^3\,dx\)
  1. Antiderivative: \(F(x) = \dfrac{x^4}{4}\).
  2. Evaluate: \(F(1) - F(-1) = \dfrac{1}{4} - \dfrac{1}{4}\).
  3. The area below the axis (left of 0) cancels the area above (right of 0).
  4. Answer: \(\;\boxed{0}\) — signed area, not total area.

The piece below the axis on \([-1,0]\) exactly cancels the piece above on \([0,1]\).

▶Example 6 — FTC Part 1 (with the chain rule)
  1. Simple case: \(\dfrac{d}{dx}\displaystyle\int_2^{x}\sqrt{t^3 + 1}\,dt = \sqrt{x^3 + 1}\) — just replace \(t\) with \(x\).
  2. Chain-rule case: \(\dfrac{d}{dx}\displaystyle\int_0^{x^2}\cos t\,dt\).
  3. Replace \(t\) with the top limit \(x^2\): \(\cos(x^2)\), then multiply by \((x^2)' = 2x\).
  4. Answer: \(\;\boxed{2x\cos(x^2)}\)
▶Example 7 — \(u\)-substitution (indefinite): \(\displaystyle\int 2x\,(x^2 + 1)^3\,dx\)
  1. Let \(u = x^2 + 1\). Then \(du = 2x\,dx\) — exactly the rest of the integrand.
  2. Rewrite: \(\int u^3\,du\).
  3. Integrate: \(\dfrac{u^4}{4} + C\).
  4. Substitute back \(u = x^2 + 1\).
  5. Answer: \(\;\boxed{\dfrac{(x^2 + 1)^4}{4} + C}\)
▶Example 8 — \(u\)-substitution (definite): \(\displaystyle\int_0^1 2x\,(x^2 + 1)^3\,dx\)
  1. Same sub \(u = x^2 + 1\), \(du = 2x\,dx\).
  2. Change the limits: when \(x = 0\), \(u = 1\); when \(x = 1\), \(u = 2\).
  3. Integral becomes \(\displaystyle\int_1^2 u^3\,du = \left[\dfrac{u^4}{4}\right]_1^2\).
  4. Evaluate: \(\dfrac{2^4}{4} - \dfrac{1^4}{4} = \dfrac{16 - 1}{4}\).
  5. Answer: \(\;\boxed{\dfrac{15}{4}}\)
▶Example 9 — Average value of \(f(x) = x^2\) on \([0, 3]\)
  1. Use \(f_{\text{avg}} = \dfrac{1}{b-a}\displaystyle\int_a^b f\,dx\) with \(a = 0,\ b = 3\).
  2. \(\displaystyle\int_0^3 x^2\,dx = \left[\dfrac{x^3}{3}\right]_0^3 = \dfrac{27}{3} = 9\).
  3. Divide by \(b - a = 3\): \(\dfrac{1}{3}(9)\).
  4. Answer: \(\;\boxed{3}\)
