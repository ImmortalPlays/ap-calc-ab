Contextual Applications of Differentiation

1. Motion along a line

If \(s(t)\) is position, then velocity and acceleration are its derivatives:

$$v(t) = s'(t), \qquad a(t) = v'(t) = s''(t)$$

Speed is \(|v(t)|\). The object is at rest when \(v(t) = 0\), and speeding up when \(v\) and \(a\) have the same sign.

2. Related rates

When several quantities change together over time, differentiate the equation relating them with respect to \(t\), then plug in the known rates. Steps: write the relationship, differentiate implicitly in \(t\), substitute, solve for the unknown rate.

3. L'Hôpital's Rule

If a limit gives the indeterminate form \(\dfrac{0}{0}\) or \(\dfrac{\infty}{\infty}\), you may differentiate top and bottom separately:

$$\lim_{x \to a}\frac{f(x)}{g(x)} = \lim_{x \to a}\frac{f'(x)}{g'(x)}$$

4. Linear approximation

Near \(x = a\), the tangent line is a good stand-in for the curve:

$$L(x) = f(a) + f'(a)(x - a)$$

▶Example 1 — Motion: \(s(t) = t^3 - 6t^2 + 9t\)
  1. Velocity: \(v(t) = s'(t) = 3t^2 - 12t + 9\).
  2. Acceleration: \(a(t) = v'(t) = 6t - 12\).
  3. At rest when \(v(t) = 0\): \(3t^2 - 12t + 9 = 3(t-1)(t-3) = 0\).
  4. Answer: the object is at rest at \(\;\boxed{t = 1 \text{ and } t = 3}\).
▶Example 2 — Related rates: an inflating spherical balloon

Air is pumped in at \(\dfrac{dV}{dt} = 100\ \text{cm}^3/\text{s}\). How fast is the radius growing when \(r = 5\) cm?

  1. Relationship: \(V = \dfrac{4}{3}\pi r^3\).
  2. Differentiate with respect to \(t\): \(\dfrac{dV}{dt} = 4\pi r^2 \dfrac{dr}{dt}\).
  3. Substitute \(\dfrac{dV}{dt} = 100\) and \(r = 5\): \(100 = 4\pi(25)\dfrac{dr}{dt}\).
  4. Solve: \(\dfrac{dr}{dt} = \dfrac{100}{100\pi}\).
  5. Answer: \(\;\boxed{\dfrac{dr}{dt} = \dfrac{1}{\pi}\ \text{cm/s}}\)
▶Example 3 — L'Hôpital's Rule: \(\lim_{x \to 0}\dfrac{\sin x}{x}\)
  1. Direct substitution gives \(\dfrac{\sin 0}{0} = \dfrac{0}{0}\) — indeterminate, so L'Hôpital applies.
  2. Differentiate top and bottom separately: \(\dfrac{(\sin x)'}{(x)'} = \dfrac{\cos x}{1}\).
  3. Now substitute \(x = 0\): \(\dfrac{\cos 0}{1} = \dfrac{1}{1}\).
  4. Answer: \(\;\boxed{1}\)
