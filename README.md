# AP Calculus AB — Study Guide

A simple website that explains each AP Calculus AB unit with clear concepts and step-by-step worked solutions.

## Status
- **Unit 1: Limits & Continuity** — live
- Units 2–8 — coming soon

## How it's built
- Plain HTML, CSS, and a little JavaScript (no build step)
- [KaTeX](https://katex.org/) renders the math notation
- Hosted with GitHub Pages

## Adding a new unit
1. Copy `units/unit-1.html` to `units/unit-2.html`
2. Replace the concepts and step-by-step solutions
3. Enable the unit's card on `index.html`
